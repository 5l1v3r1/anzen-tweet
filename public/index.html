= Anzen Tweet

== What is this?

Many cities have notification services to Keitai mail that inform you local security information
including blackout schedules after the earthquake happened on Mar. 11th 2011, but few have
Twitter accounts. Anzen Tweet is a tiny bridge Rails app that receives the mail and tweet
the contents. It can be setup easily on Heroku.

== Installation

- Create a dedicated twitter account.
- Go to http://dev.twitter.com/, then create an application. Installation script will ask you Consumer Key,
Consumer Secret, Access Token and Access Token Secret.
- Run install.rb and provide necessary information.
